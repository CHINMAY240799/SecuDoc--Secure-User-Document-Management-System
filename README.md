# Secure User & Document Management System

A production-grade microservices-based backend system built using Spring Boot.

## Architecture
- Microservices Architecture
- API Gateway
- Auth Service
- User Service
- Document Service
- Audit Service

## Tech Stack
- Java 17
- Spring Boot 3
- Spring Security
- JWT Authentication
- PostgreSQL
- Docker

## Services
- auth-service
- user-service
- document-service
- audit-service
- api-gateway

## Security
- Stateless JWT authentication
- Role-based authorization
